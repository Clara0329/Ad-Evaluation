## 📖 项目简介
本项目立足**AIGC生成图像缺乏心理维度量化评测标准**行业痛点，融合**计算机视觉+应用心理学**双学科，搭建一套自动化视觉内容心理效应量化测评平台，实现对文生图/AI生成图像从客观画质、语义匹配、视觉心理感染力三维自动打分，配套自研AIGC视觉心理测评基准数据集，填补现有AIGC评测仅关注画质、缺少人类心理影响量化的空白。

当前行业主流AIGC图像评测多聚焦清晰度、构图、文本对齐等客观指标，忽略图片带给受众情绪唤醒、心理偏好、感官冲击等隐性心理效应；本系统依托多模态大模型+目标检测+图像分割+OCR多算法协同，结合心理学量表标注规则，将主观心理感受转化0~100量化分数，可用于AIGC模型效果横向测评、广告素材心理效果检测、数字内容风险筛查三大场景。

## 🎯 研究目标
1. **基准构建**：自建AIGC视觉心理标注数据集（dataset目录），按照积极/消极/中性三类心理标签标注上万张AI生成图像，形成开源测评Benchmark；
2. **工具开发**：前后端分离Web测评系统，上传图片一键输出【客观画质分+语义匹配分+心理影响力总分】三项量化结果；
3. **落地验证**：完成多主流大模型（Qwen、DeepSeek等）生成图片横向对比实验，输出测评分析报告（experiments存放实验数据）；
4. **成果输出**：项目结题产出数据集、测评源码、实验论文、使用手册全套资料。

## ✨ 系统核心功能
### 1. 后端多模型协同评测（backend目录架构）
- `backend/qwen/`：通义千问多模态大模型，负责图文语义理解、心理情感描述、综合打分逻辑；
- `backend/yolo/`：YOLO目标检测，定位图像关键物体、情绪相关元素（恐怖元素、温馨元素等）；
- `backend/sam/`：SAM像素级图像分割，拆分局部画面，精细化局部心理特征提取；
- `backend/paddleocr/`：PaddleOCR文字识别，提取画面内嵌文案，辅助心理倾向判定；
- `backend/scoring/`：打分引擎，融合多模型输出+心理学权重系数，计算最终心理效应得分。

### 2. 前端可视化平台（frontend目录）
Web端实现图片拖拽上传、测评结果可视化、分项分数图表展示、批量图片评测、测评报告导出。

### 3. 配套数据与实验
- `dataset/`：存放自建心理标注数据集、基准测试图集、心理学标注规范文件；
- `experiments/`：消融实验、多模型对比实验脚本、实验日志、测评结果数据；
- `docs/`：立项资料、接口文档、部署手册、心理学打分细则。
